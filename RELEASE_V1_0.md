# Release v1.0

## Short Release Note

Public `v1.0` of an unofficial Pencil public skill.

This release opens the workflow layer:

- public Pencil skill
- public-safe handshake and execution discipline
- lightweight demo brief
- copyable prompts
- release boundary and privacy checks

This release does **not** include:

- official Pencil binaries
- official MCP redistribution
- private assets
- premium internal workflow packs
