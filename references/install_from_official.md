# Install From Official Source

This repository assumes the user installs Pencil from the official source on their own machine.

Recommended sequence:

1. Read the official install docs:
   - https://docs.pencil.dev/getting-started/installation
2. Read the official terms if you plan to publish around Pencil:
   - https://www.pencil.dev/terms-of-use
   - https://www.pencil.dev/eula
3. Launch Pencil locally.
4. Verify its MCP tools appear in your client.
5. Only then continue with this public workflow kit.

Do not:

- mirror the official binary in this repo
- check in activation state
- ship tokens or machine-specific setup
- imply that this repo grants rights beyond the official terms
