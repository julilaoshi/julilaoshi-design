# Official Dependency Boundary

This repository is an unofficial workflow layer.

That means:

- official Pencil stays an official dependency
- this repository stays a public wrapper around workflow, docs, and templates
- users install official Pencil themselves
- this repository should not redistribute the official product

Safe public positioning:

- "Unofficial public workflow kit"
- "Pencil-compatible"
- "Official Pencil install required"
- "Not affiliated with or endorsed by Pencil"

Unsafe positioning:

- "Official Pencil package"
- "Bundled Pencil MCP"
- "Drop-in replacement for Pencil"
- anything that suggests redistribution rights you do not have

If you ever want to:

- bundle official Pencil
- redistribute its binaries
- ship its extension package inside your own release
- sell a combined package that includes the official product

get written permission from the official vendor first.

Also remember:

- official terms can change
- release-day checks should re-read the current public terms before publishing major versions
